Tổng hợp bài làm của đội Apex:

-Toàn bộ kết quả của 10 câu hỏi trong Part 1 được lưu trong file (DATATHON)_APEX_P1code.ipynb

-Các kết quả tính toán, thống kê và biểu đồ của quá trình EDA của Part 2 được lưu trong file (DATATHON)_APEX_P2code.ipynb

-Mô hình dự đoán sales và xuất ra file submission của Part 3 được lưu trong file (DATATHON)_APEX_P3code.ipynb

-Kết quả dự đoán submit qua Kaggle được lưu trong file (DATATHON)_APEX_P3submission.ipynb

Để đảm bảo tính tái lập, người dùng cần:
Cài đặt đầy đủ các thư viên python được sử dụng trong gọi trong source code.
Tạo thư mục data/ trong cùng cấp với các file Notebook.
Sao chép toàn bộ các file .csv của cuộc thi vào thư mục này.
Cập nhật lại đường dẫn data_path phù hợp với môi trường và theo đúng thư mục lưu các file .csv trong máy tính của bạn.
Khởi chạy các cell theo đúng trình tự từ trên xuống dưới.
